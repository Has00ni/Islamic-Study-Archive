# 📌 About the Project

**Ummah Ilm Hub** is a centralized open-source platform dedicated to organizing and distributing Islamic knowledge in structured learning formats.

The goal is to provide:
- Free Islamic quizzes
- Anki flashcard decks
- Kahoot links
- Printable study materials
- JSON-based quiz files
- Topic-organized resources

All materials are categorized and structured to support self-study, teaching environments, and community learning.

---

# 🌍 Vision & Mission

### Vision
To become a global open Islamic learning library accessible to anyone, anywhere.

### Mission
- Remove login barriers for Islamic study materials.
- Provide structured, categorized learning.
- Encourage open-source collaboration.

# 🗂 Content Categories

The repository is divided into structured topics:

- **Prophets (Qasas al-Anbiya)**
- **Quran & Vocabulary**
- **Seerah (Biography of Prophet Muhammad ﷺ)**
- **Hadith Collections**
- **Aqeedah (Islamic Creed)**
- **Fiqh (Islamic Jurisprudence)**
- **Islamic History**
- **Sahabah Biographies**
- **Arabic Language for Islamic Studies**
- **Ramadan & Special Occasions**

Each category may include quizzes, flashcards, and external learning links.
